# **App Name**: PumpPal

## Core Features:

- Fuel Type UI: A UI that allows the attendant to easily select the type of fuel being dispensed.
- Payment UI: Choose between cash, card, or mobile payment options.
- History UI: A chronological list of all transactions made.
- Sales Summary Tool: Using LLM to generate Daily sales summaries.
- Settings UI: Customize tax and various settings.

## Style Guidelines:

- Primary color: Dark petrol blue (#004A7C) for reliability and trustworthiness.
- Background color: Light gray (#F0F4F7), desaturated and bright to provide a clean and neutral backdrop.
- Accent color: Soft teal (#46B1BF), to provide gentle contrast and call attention to key actions.
- Body and headline font: 'Inter', a sans-serif font for a clean and modern look.
- Use consistent and recognizable icons for navigation and actions.
- Implement a clean and intuitive layout for easy navigation.
- Subtle animations on button presses or transaction confirmations.
